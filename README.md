# Python Data Analysis Journey

This repository documents my structured learning journey in **Python for Data Analysis**.

The goal is to build strong foundations in Python and gradually move towards tools used in real-world data analysis such as:

- NumPy
- Pandas
- Matplotlib
- Seaborn

Each week contains notebooks with concepts, practice exercises, and mini projects.

More topics will be added as the learning journey progresses.
